# Pankaj Kumar's Portfolio

## 🌟 Overview
Welcome to my personal developer portfolio! This project showcases my skills, projects, and achievements as a **MERN Stack Developer** specializing in **React.js** and **full-stack web development**.

---

## 🎯 Features
- **About Me Section** – Brief intro and career journey.
- **Skills Showcase** – Highlighting my technical stack.
- **Projects Display** – Detailed breakdown of my top projects.
- **Resume Download** – Easy access to my resume.
- **Contact Form** – For quick collaboration inquiries.
- **Social Links** – Connect with me on LinkedIn, GitHub, and more.

---

## 🛠️ Tech Stack
- **Frontend:** React.js, CSS
- **Hosting:** Netlify

---

## 🚀 Setup Instructions

### 🔧 1. Clone the Repository
```bash
git clone https://github.com/pankajydv15/portfolio.git
cd portfolio
```

### 📦 2. Install Dependencies
```bash
npm install
```

### 🔥 3. Run the Portfolio
```bash
npm run dev
```
